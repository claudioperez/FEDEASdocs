General Utility Functions

Once the geometry and other relevant information about the structural model is collected in data structure [data:Model]  by the function [func:Create_SimpleModel], information about the geometry and the global DOF correspondence for a particular element can be extracted with the help of two general utility functions: Localize and ElmLenOr. All functions with input arguments [data:Model]  and [data:Data] in Table ¿tbl:StructMat? invoke these utility functions for setting up the corresponding structural matrices or vectors.

Function [func:Localize]

This function locates the element el in the [data:Model]  and returns its end node coordinates in array xyz, and the local-global DOF correspondence in vector id. The first column of the two dimensional array xyz contains the coordinates of end \(i\) of the element and the second column those of end \(j\).

Function [func:ElmLenOr]

This function returns the element length L and the direction cosines of the element \(x\)-axis in vector dcx from the end node coordinates supplied in array xyz.
